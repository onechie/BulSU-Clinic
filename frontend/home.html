backend/api/records
GETbackend/api/records

parameters :

  • id=(optional)
  • patientName=(optional)
  • if you include both, id is the first priority

response :

  • "success":bool
  • "message":"String"
  • "record":{if request has id parameter this is the response}
  • "record":[{if request has patientName and no id parameter this is the response}]
  • "records":[{array of object records}]
